# CRUD MongoDB Laravel 8
Create Read Update and Delete with Laravel 8 using Database MongoDB

## Installation
```sh
git clone https://github.com/suendri/free-crud-mongodb-laravel8.git
composer update
php artisan migrate:fresh --seed
```
